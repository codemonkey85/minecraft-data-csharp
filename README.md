# minecraft-data-csharp

A C# wrapper for [PrismarineJS/minecraft-data](https://github.com/PrismarineJS/minecraft-data).

Used for [MinecraftCommandBuilder](https://github.com/codemonkey85/MinecraftCommandBuilder).
